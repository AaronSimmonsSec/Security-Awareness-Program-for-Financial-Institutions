# 🚨 Security Findings Identified

## 🔓 Workstation Security

- [ ] Workstation unlocked while unattended
- [ ] Sensitive applications left open
- [ ] Session timeout controls bypassed
- [ ] Shared or generic credentials observed
- [ ] MFA token/device left unattended
- [ ] Unauthorized USB/removable media connected

### 📌 Observation Notes
> Example: Employee workstation remained unlocked for approximately 12 minutes while unattended during lunch period.

---

## 🗂️ Clean Desk Policy Violations

- [ ] Customer information exposed
- [ ] Confidential documents visible
- [ ] Printed reports left at printer/copier
- [ ] Passwords written on sticky notes
- [ ] Sensitive whiteboard information exposed
- [ ] Unsecured notebooks or handwritten notes

### 📌 Observation Notes
> Example: Printed account reconciliation report containing customer identifiers left unattended near shared printer.

---

## 🚪 Physical Security Concerns

- [ ] Tailgating/piggybacking observed
- [ ] Badge not visible or improperly displayed
- [ ] Restricted door propped open
- [ ] Unauthorized individual in secured area
- [ ] File cabinet/storage left unlocked
- [ ] Visitor escort procedures not followed

### 📌 Observation Notes
> Example: Individual entered secure access area behind employee without badge validation.

---

## 💻 Mobile & Remote Work Security

- [ ] Laptop unattended in common area
- [ ] Sensitive information visible on screen
- [ ] Mobile device unsecured
- [ ] Headset/speakerphone exposing confidential discussions
- [ ] Portable media improperly stored

### 📌 Observation Notes
> Example: Open laptop displaying internal financial reporting left unattended in conference room.

# 🧠 Security Awareness Reinforcement

Employees are reminded to:

- Lock workstations anytime unattended
- Protect customer and company information
- Prevent tailgating into secure areas
- Secure MFA devices and credentials
- Follow Clean Desk Policy requirements
- Report suspicious behavior immediately

# 🛡️ Professional Documentation Standards

This documentation is intended to support:
- Security awareness improvement
- Behavioral reinforcement
- Audit readiness
- Compliance alignment
- Risk reduction initiatives

Preferred terminology includes:
- Observation
- Finding
- Policy Deviation
- Security Gap
- Awareness Opportunity

Avoid emotionally charged terminology such as:
- “Offender”
- “Violation”
- “Negligence”

unless formally required by HR or Compliance procedures.
